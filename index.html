Hello World HTML breakdown

to start coding in html, you create an html file( in our example, index.html ) on a file location on your computer. after that, you need to open the file in a text editor and then input some html code( use the code example below )

index.html üìã <!DOCTYPE html> <html lang="en"> <head></head> <body> hello world </body> </html>
  • the <!DOCTYPE html> serves an indicator to the browser that we're using the html5 declaration in this html syntax.
  • you create some opening and closing <html> tags in order to contain the html code for the browser to run
  • in between the opening and closing <html> tags, you put some opening and closing <head> tags and <body> tags with the head tags coming before the body. the main reason for this is to define the actual content and metadata of the html page using the body and head tags respectively.
  • in this beginner tutorial, we aren't going to go directly into what makes the or should be put into the <head> tags but in the <body> tags just include a simple open & closed <h1> tags with the text "hello world" written into it.
